PROJECT: THE ULTIMATE TRAVEL PLANNER (CLI)
===========================================
Estimated Time: 120 Minutes
Concepts covered: Variables, Data Types, Casting, Strings, Booleans, Operators, 
Lists, Tuples, Sets, Dictionaries, If…Else, Match, While Loops, For Loops, 
Functions, Range, Math, JSON, Dates, Try…Except, User Input, File Handling.

--- TASK LIST ---

Task 1: [Variables & Casting] 
Create a variable 'budget_str' with value "1500". Cast it to a float named 'total_budget'. 
Create a variable 'destination' with the string "Paris".

Task 2: [Dates & Math] 
Import 'datetime' and 'math'. Create a date object 'travel_date' for July 15, 2026. 
Calculate 'days_until' by subtracting datetime.date.today() from travel_date. 
Use math.ceil() to round a hypothetical flight duration of 7.4 hours to 'flight_hours'.

Task 3: [Strings & Formatting] 
Create a welcome message using f-string: "Trip to {destination} in {days_until.days} days!". 
Convert 'destination' to uppercase using .upper() and print it.

Task 4: [Booleans & Operators] 
Create a boolean 'is_international' set to True. 
Create a boolean 'can_afford' by checking if total_budget >= 1000 AND is_international is True.

Task 5: [Lists & Tuples] 
Create a list 'cities_to_visit' containing: "Eiffel Tower", "Louvre", "Versailles". 
Create a tuple 'coordinates' containing (48.8566, 2.3522). 
Add "Montmartre" to the list using .append().

Task 6: [Sets] 
Create a set 'packing_list' with: "Passport", "Camera", "Charger". 
Add "Passport" again to the set. Observe how it handles duplicates. 
Add "Walking Shoes" to the set.

Task 7: [Dictionaries] 
Create a dictionary 'trip_info' with keys: "dest", "budget", "tags". 
Assign the values from your 'destination', 'total_budget', and a list ["Culture", "Food"].

Task 8: [None & User Input] 
Create a variable 'user_notes' initialized as None. 
Use input() to ask "Any special requests?" and store it in 'user_notes'.

Task 9: [Try...Except] 
Ask the user for 'additional_funds' using input(). 
Wrap the float conversion in a try...except block to handle ValueError if they type text.

Task 10: [Functions] 
Define a function 'calculate_daily_spend(total, days)' that returns total / days. 
Call it using 'total_budget' and a 'duration' of 7 days.

Task 11: [If...Else] 
Check if 'total_budget' is > 2000. If so, print "Luxury Trip", else print "Budget Trip".

Task 12: [Match...Case] 
Create a variable 'transport_mode' = "Train". 
Use a match statement to print "Fast" for "Flight", "Scenic" for "Train", and "Flexible" for "Car".

Task 13: [While Loops] 
Create a variable 'countdown' = 3. 
Use a while loop to print the countdown and decrement it until it hits 0.

Task 14: [For Loops & Range] 
Use a for loop with range(1, 4) to print "Day 1", "Day 2", "Day 3" of the itinerary.

Task 15: [Modules & JSON] 
Import the 'json' module. 
Convert the 'trip_info' dictionary into a JSON string named 'json_data' using json.dumps().

Task 16: [Write/Create Files] 
Open a file named "itinerary.txt" in 'w' mode. 
Write the 'destination' and the 'welcome_message' from Task 3 into the file. Close it.

Task 17: [Read Files] 
Open "itinerary.txt" in 'r' mode. 
Read the content into a variable 'file_content' and print it to the console.

Task 18: [File Handling - Appending] 
Open "itinerary.txt" in 'a' mode (append). 
Add a new line: "Estimated Flight Hours: " followed by 'flight_hours'.

Task 19: [Arrays] 
Import 'array'. Create an array 'expenses' of type 'd' (double) with: [120.5, 45.0, 80.75]. 
Print the first element using indexing.

Task 20: [Delete Files] 
Import 'os'. Write code that checks if a file "temp.txt" exists, and if so, deletes it. 
(Create a dummy temp.txt first to test this).
