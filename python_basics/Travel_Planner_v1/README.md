# The Ultimate Travel Planner CLI

A beginner to intermediate Python command line project that simulates planning a trip.

This project was built to practice core Python fundamentals in a single structured workflow.

## Features

- Budget handling and type casting
- Date calculations using datetime
- Math operations and rounding
- String formatting and manipulation
- Boolean logic and operators
- Lists, tuples, sets, and dictionaries
- User input handling
- Try except error handling
- Functions
- Conditional logic and match case
- While loops and for loops
- JSON conversion
- File creation, reading, appending, and deletion
- Array module usage
- OS module file checks
